# ScanPick Landing Page

[scanpick.cc](https://scanpick.cc) — Wave picking software for small warehouses.

Single-page marketing site hosted on GitHub Pages. No build step — pure HTML + Tailwind CSS (CDN).

## Updating

Edit `index.html`, commit, push. GitHub Pages auto-deploys from `main`.

## Custom Domain

`scanpick.cc` is configured via `CNAME` file. DNS must point to GitHub Pages:

```
scanpick.cc  CNAME  timdehof.github.io.
```

## Stripe Checkout

Pricing "Buy Now" links are `#` placeholders. Replace with real Stripe Checkout URLs before launch.
